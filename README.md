# Dark mode for Stoplight Elements

To use dark mode switching (OS-dependant) with Stoplight Elements, just place these two lines of code after script and stylesheet [provided by Stoplight Elements](https://docs.stoplight.io/docs/elements/a71d7fcfefcd6-elements-in-html):

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/sirkadirov/stoplight-elements-auto-theme@main/styles.css">
<script src="https://cdn.jsdelivr.net/gh/sirkadirov/stoplight-elements-auto-theme@main/script.js"></script>
```
